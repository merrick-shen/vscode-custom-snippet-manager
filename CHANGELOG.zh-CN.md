## [1.0.0] - 2026-05-01

### 新增

- **片段管理**：支持创建、编辑和删除代码片段，包含名称、触发前缀、代码内容、描述和适用语言字段
- **智能补全**：基于 CompletionItemProvider 的前缀触发补全，支持 SnippetString 制表位光标跳转（`$1`、`$2`、`$0`）
- **侧边栏界面**：基于 Webview 的片段列表，支持模糊搜索（Fuse.js）和语言筛选
- **编辑器界面**：基于 Webview 的表单，支持 CodeMirror 6 语法高亮和输入校验
- **语言图标**：基于 Iconify 的语言图标，显示在下拉菜单和片段列表项中
- **国际化**：完整的简体中文 / 繁體中文 / English / 日本語 / 한국어 界面支持，下拉菜单切换语言，偏好跨会话持久保存
- **导入与导出**：导出代码片段为 JSON 文件（`code_snippet_config_YYYYMMDD_HHMMSS.json`），包含版本元数据；从 JSON 文件导入，支持数据验证、安全检查和重复处理（覆盖/跳过/合并）
- **排序功能**：按添加日期排序，支持正序/倒序切换；排序相关字段在导出时自动剔除
- **命令**：
  - `新建片段` — 打开编辑器创建新片段
  - `打开片段库` — 聚焦侧边栏面板
  - `插入片段` — 基于 QuickPick 的片段插入（`Ctrl+Alt+I`）
  - `触发片段补全` — 触发自动补全建议（`Ctrl+Alt+Space`）
  - `保存到片段库` — 通过右键菜单将选中代码保存为片段（自动识别语言）
- **右键菜单**：编辑器右键菜单中添加"插入片段"和"保存到片段库"选项
- **数据存储**：使用 VS Code 的 `globalStorageUri` 进行本地 JSON 文件持久化
- **删除确认**：自定义弹窗对话框，防止误删
- **错误处理**：侧边栏和编辑器视图中带有自动消失动画的错误提示条
